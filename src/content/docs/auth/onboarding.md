Setelah pemilik bimbingan belajar (owner) mendaftarkan akun baru, sistem akan menampilkan panduan setup interaktif multi-langkah (*Onboarding Wizard*). Langkah ini wajib diselesaikan untuk menginisialisasi sistem dasar sebelum dasbor utama dapat digunakan.

---

## Langkah-Langkah Persiapan Awal

Panduan inisialisasi ini terdiri dari 4 langkah mudah:

1.  **Langkah 1: Identitas Lembaga**:
    *   **Nama Bimbel**: Nama resmi instansi Anda yang akan tampil di header aplikasi dan halaman pendaftaran.
    *   **Deskripsi Bimbel**: Penjelasan singkat atau slogan bimbel Anda.
    *   **Alamat Web Pendaftaran (Slug)**: Menentukan tautan unik pendaftaran mandiri (contoh: mengisi `bimbel-cerdas` akan menghasilkan link `portalpintar.id/daftar/bimbel-cerdas`).
2.  **Langkah 2: Cabang Pertama**:
    *   **Nama Cabang**: Nama lokasi cabang fisik pertama Anda (contoh: *Cabang Pusat* atau *Cabang Antapani*).
    *   **Alamat Cabang**: Alamat fisik cabang untuk mempermudah pengaturan lokasi kelas siswa.
3.  **Langkah 3: Tahun Ajaran**:
    *   Menentukan nama tahun ajaran aktif perdana (contoh: `2024/2025 Ganjil`) serta tanggal mulai dan tanggal berakhirnya periode tersebut.
4.  **Langkah 4: Program Studi Pertama**:
    *   Membuat satu program belajar utama (contoh: *Matematika SD*) untuk memicu sistem kelas.

Setelah menekan tombol **"Selesai & Masuk Dasbor"**, konfigurasi awal disimpan ke database dan pengelola akan langsung dialihkan ke halaman utama Dasbor Admin (`/dashboard`).
